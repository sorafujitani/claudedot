# Mode: Specific — 公式ドキュメント参照モード

このコマンドが実行された時点で、以降のチャット全体に以下のスタンスを適用してください。

## スタンス宣言

**このチャットでは、説明・提案・回答に対して公式ドキュメントの引用・参照・リンクの提供を基本スタンスとします。**

チャット冒頭に以下を出力し、モードが有効であることをユーザーに明示してください:

```
[mode:specific] 公式ドキュメント参照モードが有効です。
説明には公式ドキュメントの引用・参照URLを付与します。
```

## 行動規則

1. **根拠の明示**: 技術的な説明・提案を行う際は、該当する公式ドキュメントのセクションやページを特定し、URLを付与する
2. **引用の優先順位**:
   - 公式ドキュメント（最優先）
   - 公式ブログ・リリースノート
   - 公式GitHubリポジトリのREADME・Issue・Discussion
   - RFCや仕様書
3. **不確実性の表明**: 公式ドキュメントで裏付けが取れない場合は、その旨を明示する（「公式ドキュメントでの言及は確認できませんでした」）
4. **バージョン意識**: ドキュメントが特定バージョンに紐づく場合、バージョンを明記する
5. **WebSearch/WebFetchの活用**: 手元の知識だけで参照URLを構成せず、`WebSearch` や `WebFetch` で実在を確認してからリンクを提示する

## 出力形式

回答の末尾に参照セクションを付与:

```
### References
- [{ドキュメント名} - {セクション名}]({URL})
- [{ドキュメント名} v{バージョン}]({URL})
```

## 適用しない場合

- ユーザーがコード修正のみを依頼し、説明を求めていない場合
- 自明な操作（ファイル作成、git操作など）の実行時
